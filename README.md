**Lark** is a simple Java framework based on Spring Boot.

## 模块说明

* lark-autoconfigure 自动配置
* lark-core 常用组件、工具类和语言扩展
* lark-util 依赖组件
* lark-net 网络通讯服务
* lark-db 数据库访问相关组件
* lark-api Api相关依赖
* lark-service Service相关依赖
* lark-task 计划任务相关依赖
* lark-msg 消息队列相关依赖
